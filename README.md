# Starfield Simulation

## Prerequisites
In order to compile this project you will need to:
* have [SFML](https://www.sfml-dev.org/) installed on your system. If you don't know how to do it see [this link](https://www.sfml-dev.org/tutorials/2.5/#getting-started).
* have [CMake](https://cmake.org/) installed
